Description

Hangman is a simple game for one player. The computer will choose a word, phrase or sentence and the player tries to guess it by suggesting letters or numbers, within a certain number of guesses.

Sample usage

Just run the game and start guessing. You can specify options to extend the funcitonality of the program, but it's up to you how you'll implement it and what options you'll decide to use. In general, the program should be able to run with:

[test@test ]$ ./hangman
Let's guess the word: _ _ _ _ _ _ _
