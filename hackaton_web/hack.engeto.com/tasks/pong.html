Description

Pong is a two-dimensional sports game that simulates table tennis. The player(s) control a paddle by moving it vertically across the left (or right) side of the screen. The game can be implemented as multiplayer (2 humans) or single-player (the other guy is the pc :)), each player controlling a his paddle on the opposing side. Players use the paddles to hit a ball back and forth. The aim is for each player to reach eleven points before the opponent; points are earned when one fails to return the ball to the other - the ball gets behind his paddle.

In order to implement this game you will need to learn some basics of pygame library. With its help you will be able to draw the game objects and run the game loop

If you never worked with pygame, you can use this quick guide to get started
